bent_plume_model.Particle.run_sbm¶

Particle.run_sbm(profile)[source][source]¶

Set up and run the single_bubble_model to track outside plume

Continues the simulation of the particle outside the plume using the single_bubble_model. The object containing the simulation and simulation results will be added to the attributes of this Particle object.

Parameters:

profile : ambient.Profile object

Ambient CTD data for the model simulation
