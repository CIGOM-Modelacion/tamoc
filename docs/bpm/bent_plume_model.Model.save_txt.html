bent_plume_model.Model.save_txt¶

Model.save_txt(base_name, profile_path, profile_info)[source][source]¶

Save the state space in ascii text format for exporting

Save the state space (dependent and independent variables) in an ascii text file for exporting to other programs (e.g., Matlab).

Parameters:

base_name : str

Base file name for the output file. This method will append the .txt file extension to the data output and write a second file with the header information called <base_name._header.txt.

profile_path : str

String stating the file path to the ambient profile data relative to the directory where fname will be saved.

profile_info : str

Single line of text describing the ambient profile data.

See also

stratified_plume_model.Model.save_txt, single_bubble_model.Model.save_txt

Notes

The output will be organized in columns, with each column as follows:

0 : time (s) 1-n : state space

The header to the output file will give the extact organization of each column of the output data.

These output files are written using the numpy.savetxt method.
