Single Bubble Model: Droplet simulations¶

Use the TAMOC single_bubble_model to simulate the trajectory of a light oil droplet rising through the water column. This script demonstrates the typical steps involved in running the single bubble model.

It uses the ambient data stored in the file /test/output/test_bm54.nc, created by the test_ambient module. Please make sure all tests have passed before running this script or modify the script to use a different source of ambient data.
