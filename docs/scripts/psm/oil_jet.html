Particle Size Models: Pure Oil Jet¶

Use the TAMOC particle_size_models module to simulate a laboratory scale pure oil jet into water. This script demonstrates the typical steps involved in using the particle_size_models.PureJet object, which requires specification of all of the fluid properties of the jet.
