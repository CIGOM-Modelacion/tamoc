bent_plume_model.Particle.run_sbm¶

Particle.run_sbm(self, profile)[source]¶

Run the single_bubble_model to track particles outside the plume

Continues the simulation of the particle is outside of the plume using the single_bubble_model. The object containing the simulation and simulation results will be added to the attributes of this Particle object.

Parameters
profileambient.Profile object

Ambient CTD data for the model simulation
