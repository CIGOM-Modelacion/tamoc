particle_size_models.Model¶

class particle_size_models.Model(profile, oil, m, z0, Tj=None)[source]¶

Master lass object for computing bubble and droplet size distributions

This model class contains handles the interface to the ModelBase, allowing particle size distributions to be easily computed given a dbm.FluidMixture description of the released fluids. This is the main class object that should be used to compute bubble and droplet size distributions for the plume models in TAMOC.
