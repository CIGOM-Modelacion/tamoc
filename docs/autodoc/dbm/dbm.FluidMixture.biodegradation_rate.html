dbm.FluidMixture.biodegradation_rate¶

FluidMixture.biodegradation_rate(self, t, lag_time=True)[source]¶

Determine the biodegradation rate constant

Returns the first-order biodegradation rate constant after the simulation time exceeds the bacterial community response lag time.

Parameters
tfloat

current simulation time (s)

lag_timebool, default = True

flag indicating whether the biodegradation rates should include a lag time (True) or not (False). Default value is True.

Returns
k_biondarray, size (nc)

first-order biodegradation rate constants (1/s)

Notes

The user provides both the first-order rate constants and the constant lag times in the user_data provided to the model or using the results in the TAMOC dataset ./data/BioData.csv.
