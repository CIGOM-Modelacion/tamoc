blowout.Blowout.simulate¶

Blowout.simulate(self)[source]¶

Run a bent_plume_model simulation for the present conditions

Calls the bent_plume_model.Model.simulate() method with the initial conditions presently stored in the class object. This method does not have any input parameters and does not return a value. After the simulation is run, the bent_plume_model.Model object will store the solution, and this object is stored as the bpm attribute of the present class.
