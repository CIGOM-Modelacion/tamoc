blowout.create_ambient_profile¶

blowout.create_ambient_profile(data, labels, units, comments, nc_name, summary, source, sea_name, p_lat, p_lon, p_time, ca=[])[source]¶

Create an ambient Profile object from given data

Create an ambient.Profile object using the given CTD and current data. This function performs some standard operations to this data (unit conversion, computation of pressure, insertion of concentrations for dissolved gases, etc.) and returns the working ambient.Profile object. The idea behind this function is to separate data manipulation and creation of the ambient.Profile object from fetching of the data itself.

Parameters
datanp.array

Array of the ambient ocean data to write to the CTD file. The contents and dimensions of this data are specified in the labels and units lists, below.

labelslist

List of string names of each variable in the data array.

unitslist

List of units as strings for each variable in the data array.

commentslist

List of comments as strings that explain the types of data in the data array. Typical comments include ‘measured’, ‘modeled’, or ‘computed’.

nc_namestr

String containing the file path and file name to use when creating the netCDF4 dataset that will contain this data.

summarystr

String describing the simulation for which this data will be used.

sourcestr

String documenting the source of the ambient ocean data provided.

sea_namestr

NC-compliant name for the ocean water body as a string.

p_latfloat

Latitude (deg)

p_lonfloat

Longitude, negative is west of 0 (deg)

p_timenetCDF4 time format

Date and time of the CTD data using netCDF4.date2num().

calist, default=[]

List of gases for which to compute a standard dissolved gas profile; choices are ‘nitrogen’, ‘oxygen’, ‘argon’, and ‘carbon_dioxide’.

Returns
profileambient.Profile

Returns an ambient.Profile object for manipulating ambient water column data in TAMOC.
