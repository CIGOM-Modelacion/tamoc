dispersed_phases.bf_average¶

dispersed_phases.bf_average(particles, rho, g, rho_r, parm)[source]¶

Compute a buoyancy-flux-weighted average of parm

Computes a weighted average of the values in parm using the kinematic buoyancy flux of each particle containing parm as the weight in the average calculation.

Parameters
particleslist of Particle objects

List of SingleParticle, PlumeParticle, or bent_plume_model.Particle objects describing each dispersed phase in the simulation

rhofloat

Local density of ambient fluid outside plume (kg/m^3).

gfloat

Acceleration of gravity (m/s^2).

rho_rfloat

Model reference density (kg/m^3).

parmndarray

Numpy array of parameters to average, one value for each dispersed phase entry (same as elements in parm).

Returns
parm_avefloat

The weighted average of parm.
