smp.outer_surf¶

smp.outer_surf(yi, p)[source]¶

Compute the initial condition for the outer plume at the sea surface

Computes the initial conditions for the first outer plume segment after the inner plume impinges on the free surface of the water body. It is assumed that the inner plume had significant volume flux and that this first outer plume segment will be viable.

Parameters
yistratified_plume_model.InnerPlume object

Object for manipulating the inner plume state space

pModelParams object

Object containing the fixed model parameters for the stratified plume model.

Returns
z0float

Initial depth of the outer plume segment (m).

y0ndarray

Initial dependent variables state space for the outer plume segment.
