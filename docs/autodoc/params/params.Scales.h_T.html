params.Scales.h_T¶

Scales.h_T(self, z0)[source]¶

Compute the trap height for the lowest intrusion

Compute the intrusion layer height above the bottom for the lowest intrusion based on the correlations in Socolofsky and Adams (2005).

Parameters
z0float

Depth to evaluate the trap height (m)

Returns
h_Tfloat

Plume trap height for the first intrusion measured in height above the bottom (m)
