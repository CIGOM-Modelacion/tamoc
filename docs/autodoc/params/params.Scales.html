params.Scales¶

class params.Scales(profile, particles)[source]¶

Compute the characteristic scales for a TAMOC simulation

Compute the governing non-dimensional parameters defined in Socolofsky and Adams (2002, 2005) to provide guidance on model selection for the TAMOC modeling suite.

Parameters
profileambient.Profile object

Ambient CTD data for the model simulation

particleslist

List of stratified_plume_model.Particle objects that define all of the dispersed phases in the simulation
