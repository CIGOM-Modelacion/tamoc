dbm_utilities.load_tamoc_oil¶

dbm_utilities.load_tamoc_oil(substance)[source]¶

Create a dbm.FluidMixture object from the given TAMOC composition

Create a dbm.FluidMixture object that contains the given composition and pseudo-component mass fraction.

Parameters
substancedict

A Python dictionary that contains keywords:

compositionlist

List of string names that correspond to compounds in the TAMOC chemical properties database

masseslist or ndarray

Array of masses (kg) for each pseudo-component in the TAMOC composition list

Returns
compositionlist

List of strings containing the names of the oil components in the dead oil from the Adios database.

mass_fracnp.array

An array of mass fractions for all compounds in the dead oil from the Adios database (kg).

user_datadict

A dictionary of chemical property data in the format expected by the tamoc.dbm module FluidMixture or FluidParticle objects.

deltanp.array (len M, len M)

Array of binary interaction coefficients

unitsdict

List of units corresponding to the dictionary of user_data
