dbm_utilities.sequence_names¶

dbm_utilities.sequence_names(sara_names, name)[source]¶

Add sequential counters to the pseudo-component name in sara_names

Parameters
sara_nameslist

List of strings containing the names of each pseudo-component in an oil based on SARA analysis.

namestr

Name of the pseudo-component to edit (‘Saturates’ or ‘Aromatics’)

Notes

Since the sara_names in the NOAA OilLibrary are stored in a list, this function makes use of the fact that Python lists are mutable; hence, changes made to sara_names in this function will also be reflected for that variable in the calling function or program.
