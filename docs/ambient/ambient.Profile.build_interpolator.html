ambient.Profile.build_interpolator¶

Release:0.1
Date:June 03, 2015
Profile.build_interpolator()[source][source]¶

Build the interpolator function from the netCDF dataset

Extract the raw data from the netCDF dataset using the required variables T, S, and P plus any variables requested by the user through self.chem_names and store the resulting data in z and y. Coarsen the raw dataset to the level specified by self.err using the coursen function and put the resulting reduced dataset into the interpolation function f.

Notes

This function is responsible for creating the object attributes z, y, f_names, f_units, and f.

This method is called by the object initializer and by the object append method. It should not be called directly by the user.
