ambient.Profile.get_values¶

Release:1.1
Date:Nov 08, 2017
Profile.get_values(z, names)[source][source]¶

Return values for the variables listed in names interpolated to the depth given by z

Parameters:

z : float or ndarray

Depth(s) at which data are desired. If the value of z lies outside the valid_min or valid_max values of z in the netCDF dataset, the values at the nearest boundary are returned.

names : string list

List of variable names (e.g., temperature, oxygen, etc.) for which the interpolated data are desired. If the parameter name is not present in the database, a value of zero is returned.
Returns:

yp : ndarray

An array of values sorted in the same order as names and interpolated at the depth(s) given by z. If z is a row vector, yp contains z in the first column and the other variables in the adjacent columns.
