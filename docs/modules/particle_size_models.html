Overview of Particle Size Models Module¶

Release

2.1

Date

Jun 05, 2020

Particle Size Models¶

Compute particle size distributions for jets of oil and gas

This module provides an interface to available empirical models for bubble and droplet size distributions for jet releases. The empirical model functions are in psf.py; this module provides an object-oriented interface to these functions. Gas bubble size distributions can be created from:

  • Li et al. (2017)

  • Wang et al. (2018)

Oil droplet size distributions can be created from:

  • Johansen et al. (2013)

  • Li et al. (2017)

All models support log-normal and Rosin-Rammler distributions.

Notes¶

The particle size computational algorithms are contained in the module

psf.py - Particle Size Fuctions

This module is a function library used by the objects in this module. In general, the psf module can be replace by any module having the same application programming interface, whether programmed in Python, or another language and wrapped in Python.

See Also¶

To date, only simple, analytical equations have been implemented in the partice size functions. More complex, physics-based models using a population dynamic approach are also often used. These may be added in the future. See, for example:

Zhao, L., Boufadel, M. C., Socolofsky, S. A., Adams, E., King, T., and Lee, K. (2014). “Evolution of droplets in subsea oil and gas blowouts: Development and validation of the numerical model VDROP-J.” Mar Pollut Bull, 83(1), 58-69.

Base Model Object¶

The base model object handles computing fluid properties and selecting appropriate particle size models. It is documented in the following:

ModelBase.get_de_max(self, fp_type)

Report the maximum stable particle size of a fluid at the release

Create plots of the bubble and droplet size distribution

Pure Jet Object¶

When only one phase (liquid or gas) is emitted from an orifice, properties for another phase should not be required. This is solved using the PureJet object, described in the following:

PureJet.get_d50(self[, fp_type])

External Utilities¶

Each of the particle size models themselves are contained in separate functions defined in the psf module. While these functions could be called directly for scripting purposes, it is recommended that the objects defined above be used instead. These utilities are the following:

mass2vol(m, rho)

Convert a mass or mass flux to a volume or volume flux

rosin_rammler(nbins, d50, k, alpha)

Return the volume size distribution from the Rosin-Rammler distribution

log_normal(nbins, d50, sigma)

Return the volume size distribution from the Log-normal distribution

ln2rr(d50, sigma)

Convert the parameters of a log-normal distribution to Rosin-Rammler

rr2ln(d50, k, alpha)

Convert the parameters of a Rosin-Rammler distribution to log-normal

rosin_rammler_fit(d50, d_max[, alpha])

Return d_50, k, and alpha for the Rosin-Rammler distribution

log_normal_fit(d50, d_max[, sigma])

Return d_50 and sigma for the log-normal distribution

de_max_oil(rho_p, sigma, rho)

Calculate the maximum stable oil droplet size

grace(rho_c, rho_d, mu_c, mu_d, sigma[, fp_type])

sintef(d0, m_gas, rho_gas, m_oil, rho_oil, …)
