Params Module¶

Calculate the governing scales and determine the appropriate simulation model

This module calculates the governing non-dimensional parameters defined in papers by Socolofsky and Adams (2002, 2005) and Socolofsky et al. (2011) and uses this information to recommend the appropriate simulation model.
