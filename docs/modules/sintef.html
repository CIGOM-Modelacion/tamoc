Sintef¶

Evaluate initial bubble and droplet sizes from the SINTEF model

This module is deprecated and has been replaced by the particle_size_models and psf modules. In order to retain compatibility with previous versions, we retain the original API of the sintef model, but replace all calculations with calls to the appropriate functions in particle_size_models and psf.

Module Functions¶

This module contains functions for computing particle size distributions using the SINTEF model. This module has been deprecated, and you should use the objects in the particle_size_models instead. In order to retain backward compatibility with scripts developed using previous versions of TAMOC, we have retained this module and edited each function to call the associated function in the psf module of utilities used by the particle_size_models objects. Functions in the SINTEF module are:

modified_We_model(D, rho_gas, m_gas, mu_gas, …)

This function is deprecated: Use psf.sintef() instead.

de_max(sigma, rho_p, rho)

This function is deprecated: Use psf.de_max_oil() instead.

de_50(U, D, rho_p, mu_p, sigma, rho)

This function is deprecated: Use psf.sintef_d50() instead.

rosin_rammler(nbins, d50, md_total, sigma, …)

This function is deprecated: Use psf.rosin_rammler() instead.
