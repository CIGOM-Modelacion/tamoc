Model Share¶

Provides functions that are common to all models in the TAMOC suite

This module defines functions that perform standard tasks for the modeling modules in the TAMOC suite. These include standard authoring of output files and reading ambient data from output files.

Notes¶

These class objects and helper functions may be used throughout the TAMOC modeling suite.

Module Functions¶

The functions contained in this module are listed as follows:

tamoc_nc_file(fname, title, summary, source)

Write the header meta data to an netCDF file for a TAMOC output
