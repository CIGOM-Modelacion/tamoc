params.Scales.h_S¶

Release:0.1
Date:June 23, 2015
Scales.h_S(z0, u_inf)[source][source]¶

Compute the cross-flow separation height

Compute the height above the bottom where the cross-flow causes separation of the entrained plume fluid from the plume based on the correlation in Socolofsky and Adams (2002).

Parameters:

z0 : float

Depth to evaluate the trap height (m)

u_inf : float

Magnitude of the local ambient cross-flow velocity (m/s)
Returns:

h_S : float

Cross-flow separation height measured in height above the bottom (m)
