params.Scales.lambda_1¶

Release:1.1

Scales.lambda_1(z0, n)[source][source]¶

Compute the spreading ratio of particle n at the given depth

Compute the spreading ratio lambda_1 for particle n at the given depth z0 from the correlation equations in Socolofsky and Adams (2005).

Parameters:

z0 : float

Depth to evaluate the trap height (m)

n : int

Index to the self.Particle object for which the spreading ratio should be calculated
Returns:

lambda_1 : float

Dispersed-phase spreading ratio lambda_1 for the selected dispersed-phase particle.
