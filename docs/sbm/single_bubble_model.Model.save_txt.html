single_bubble_model.Model.save_txt¶

Release:1.1
Date:Nov 08, 2017
Model.save_txt(base_name, profile_path, profile_info)[source][source]¶

Save the state space in ascii text format for exporting

Save the state space (dependent and independent variables) in an ascii text file for exporting to other programs (e.g., Matlab).

Parameters:

base_name : str

The main name of the output file. This method writes two files: the data are stored in base_name.txt, and the header information describing each row of data are saved in base_name_header.txt.

profile_path : str

String stating the file path relative to the directory where the output will be saved to the ambient profile data.

profile_info : str

Text describing the ambient profile data (less than 60 characters).

Notes

The output data will be organized in columns, with each column as follows:

0 : Time (s) 1 : Depth (m) 2 : (n-1) : Masses of the particle components (kg) n : Heat (m_p * cp * T) (J)

A header will be written at the top of the file with the specific details for that file.

The file is written using the numpy.savetxt method.
