dbm.FluidParticle.density¶

Release:0.1
Date:July 08, 2015
FluidParticle.density(m, T, P)[source][source]¶

Compute the particle density (kg/m^3) of the fluid in the phase given by fp_type.

Parameters:

m : ndarray, size (nc)

masses of each component in the particle (kg)

T : float

particle temperature (K)

P : float

particle pressure (Pa)
Returns:

rho_p : float

density of the fluid particle (kg/m^3)

Notes

Uses the density method in the FluidMixture object, but only returns the value for the phase given by fp_type.
