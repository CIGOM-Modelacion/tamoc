dbm.FluidParticle.interface_tension¶

Release:1.1
Date:October 31, 2017
FluidParticle.interface_tension(m, T, S, P)[source][source]¶

Computes the interfacial tension between the particle and water

Computes the interfacial tension between the particle and water. This method uses equations in Danesh (1998).

Parameters:

m : ndarray, size (nc)

masses of each component in the particle (kg)

T : float

particle temperature (K)

S : float

salinity of the ambient seawter (psu)

P : float

particle pressure (Pa)
Returns:

sigma_p : float

interfacial tension (N/m)

Notes

Uses the density method in the FluidMixture object, but only returns the value for the phase given by fp_type.
