dbm.FluidParticle.fugacity¶

Release:0.1
Date:March 27, 2015
FluidParticle.fugacity(m, T, P)[source]¶

Compute the particle fugacities (Pa) of the fluid in the phase given by fp_type.

Parameters :

m : ndarray, size (nc)

masses of each component in a particle (kg)

T : float

particle temperature (K)

P : float

particle pressure (Pa)
Returns :

fk : ndarray, size (nc)

fugacities of each component of the fluid particle (Pa)

Notes

Uses the fugacity method in the FluidMixture object, but only returns the values for the phase given by fp_type.
