dbm.FluidMixture.viscosity¶

Release:0.1
Date:June 23, 2015
FluidMixture.viscosity(m, T, P)[source][source]¶

Computes the dynamic viscosity of the gas/liquid mixture.

Computes the dynamic viscosity of gas and liquid using correlation equations in Pedersen et al. (2014) “Phase Behavior of Petroleum Reservoir Fluids”, 2nd edition, chapter 10. This function has been tested for non-hydrocarbon mixtures (oxygen, carbon dioxide) and shown to give reasonable results; hence, the same equations are used for all mixtures.

Parameters size (nc)
masses of each component in a mixture (kg)
T : float
mixture temperature (K)
P : float
mixture pressure (Pa)
Returns:

mu_p : ndarray, size (2)

dynamic viscosity for gas (row 1) and liquid (row 2) (Pa s)
