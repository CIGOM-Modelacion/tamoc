dbm.InsolubleParticle.particle_shape¶

Release:0.1
Date:September 11, 2013
InsolubleParticle.particle_shape(m, T, P, Sa, Ta)[source]¶

Determine the shape of an inert fluid particle from the properties of the particle and surrounding fluid.

Parameters :

m : float

mass of the inert fluid particle (m)

T : float

particle temperature (K)

P : float

particle pressure (Pa)

Sa : float

salinity of ambient seawater (psu)

Ta : float

temperature of ambient seawater (K)
Returns :

A tuple containing: :

shape : integer

1 - sphere, 2 - ellipsoid, 3 - spherical cap, 4 - rigid

de : float

equivalent spherical diameter (m)

rho_p : float

particle density (kg/m^3)

rho : float

ambient seawater density (kg/m^3)

mu : float

ambient seawater dynamic viscosity (Pa s)

sigma : float

interfacial tension (N/m)

Notes

We use the particle temperature to calculate properties at the interface (e.g., to calculate the interfacial tension) and the ambient temperature for properties of the bulk continuous phase (e.g., density and viscosity).

Uses the Fortran subroutines in ./src/dbm_phys.f95.
