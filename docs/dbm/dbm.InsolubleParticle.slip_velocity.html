dbm.InsolubleParticle.slip_velocity¶

Release:0.1
Date:October 13, 2015
InsolubleParticle.slip_velocity(m, T, P, Sa, Ta, status=-1)[source][source]¶

Compute the slip velocity (m/s) of an inert fluid particle.

Parameters:

m : float

mass of the inert fluid particle (kg)

T : float

particle temperature (K)

P : float

particle pressure (Pa)

Sa : float

salinity of ambient seawater (psu)

Ta : float

temperature of ambient seawater (K)

status : int

flag indicating whether the particle is clean (status = 1) or dirty (status = -1). Default value is -1.
Returns:

us : float

slip velocity of the inert particle (m/s)

Notes

Uses the Fortran subroutines in ./src/dbm_phys.f95.
