dbm.InsolubleParticle.viscosity¶

Release:0.1
Date:March 27, 2015
InsolubleParticle.viscosity(T)¶

Computes the dynamic viscosity of the liquid if applicable.

Computes the dynamic viscosity of gas and liquid using correlation equations in McCain (1990).

Parameters :

T : float

mixture temperature (K)
Returns :

mu_p : ndarray, size (2)

dynamic viscosity (Pa s)
