simp.main_ic¶

Release:0.1
Date:March 12, 2014
simp.main_ic(profile, particles, p, z0, R)[source]¶

Compute the initial conditions for the inner plume state space

Compute the initial conditions at the release location for the inner plume state space

Parameters :

profile : ambient.Profile object

The ambient CTD object used by the single bubble model simulation.

particles : list of Particle objects

List of Particle objects containing the dispersed phase initial conditions

p : stratified_plume_model.ModelParams object

Object containing the fixed model parameters for the stratified plume model.

z0 : float

Depth of the release point (m)

R : float

Radius of the release port (m)
