seawater.density¶

Release:0.1
Date:November 06, 2013
seawater.density(T, S, P)[source]¶

Computes the density of seawater from Gill (1982)

Computes the density of seawater using the equation of state in Gill (1982), Ocean-Atmosphere Dynamics, Academic Press, New York. The equations for this code are taken from Appendix B in Crounse (2000).

Parameters :

T : float

temperature (K)

S : float

salinity (psu)

P : float

pressure (Pa)
Returns :

rho : float

seawater density (kg/m^3)
